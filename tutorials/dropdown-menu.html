Overview

The Dropdown Menu plugin allows adding a configurable dropdown menu to the table's column headers.
The dropdown menu acts like the Context Menu, but is triggered by clicking the indicator in the header.

Quick setup

To enable the plugin, simply set the dropdownMenu property to true, when initializing Handsontable.
