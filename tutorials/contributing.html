Your contributions to the project are very welcome. If you would like to fix a bug or propose a new feature, you can submit a Pull Request.

To help us merge your Pull Request, please make sure you follow these points:

  1. Please make your fix on a separate branch. This makes merging much easier.
  2. Do not edit files handsontable.js, handsontable.css, handsontable.full.js, handsontable.full.css. Instead, try to edit files inside the src/ directory and then use grunt to make a build. Find more details on Custom Build page.
  3. Very important: For any change that you make, please add test cases in tests/jasmine/spec/ or src/3rdparty/walkontable/test/jasmine/spec/. This helps us understand the issue and make sure that it will stay fixed forever. See Testing for details.
  4. Very important: Please review our Seven Principles for instructions on how to maintain a fork and submit patches.
  5. Describe the problem in the pull request description.

Your commitment is much appreciated, thanks!
